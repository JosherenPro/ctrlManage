# Reporting et Export

## Objectif

Fournir une visibilité exploitable sur l'assiduité.

## Rapports MVP

- Présences par session
- Taux de présence par cours
- Historique d'un étudiant
- Vue globale par classe

## Exports

- `CSV` pour retraitement administratif
- `PDF` pour diffusion et archivage

## Indicateurs utiles

- Taux de présence
- Nombre d'absences
- Nombre de retards
- Évolution sur une période
