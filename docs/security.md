# Sécurité et Conformité

## Principes

- Moindre privilège
- Validation stricte des entrées
- Traçabilité
- Protection des données personnelles

## Mesures minimales

- Chiffrement en transit via `HTTPS`
- Hashage sécurisé des secrets applicatifs
- Validation serveur des payloads
- Limitation de débit sur les endpoints sensibles
- Journalisation des actions critiques

## Menaces à couvrir

- Usurpation de rôle
- Rejeu de QR code
- Accès non autorisé aux données de présence
- Exports abusifs

## Exigences d'audit

- Qui a fait l'action
- Quand l'action a été réalisée
- Quelle entité a été modifiée
- Quelle opération a été exécutée
