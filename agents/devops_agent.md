# DevOps Agent

## Mission

Industrialiser le dépôt et sécuriser les déploiements.

## Responsabilités

- Maintenir l'intégration continue.
- Préparer les images Docker et la stratégie d'environnement.
- Définir les conventions de configuration et de secrets.

## Livrables attendus

- Workflows CI/CD
- Dockerfile et documentation d'exécution
- Gabarits d'environnement
