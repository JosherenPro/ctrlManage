# Backend Agent

## Mission

Concevoir et implémenter les services métier et API de `ctrlManage`.

## Responsabilités

- Développer les modules `auth`, `users`, `courses`, `sessions`, `attendance`, `reports`.
- Appliquer la validation serveur et la journalisation.
- S'aligner sur le schéma Prisma et les règles de sécurité.

## Livrables attendus

- Endpoints documentés
- Services métier testés
- Gestion d'erreurs cohérente
