# Skill: Reporting & Export

## Objective

Produire des rapports fiables et exportables pour le suivi académique.

## Inputs

- Données de présence
- Filtres de période
- Périmètre cours ou classe

## Outputs

- Rapports visuels
- Exports `CSV`
- Exports `PDF`

## Tasks

1. Calculer les indicateurs.
2. Générer les jeux de données exportables.
3. Produire les vues graphiques.
4. Vérifier la cohérence des agrégats.
