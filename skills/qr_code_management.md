# Skill: QR Code Management

## Objective

Generate and validate secure QR codes for attendance tracking.

## Inputs

- `session_id`
- `course_id`
- `expiration_time`

## Outputs

- `qr_code`
- `attendance_record`

## Tools

- `qrcode`
- `html5-qrcode`

## Tasks

1. Generate a unique QR code.
2. Set expiration time.
3. Validate scanned data.
4. Record attendance.

## Deliverables

- Backend service
- Frontend scanner
- API endpoints
- Documentation
