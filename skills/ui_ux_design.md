# Skill: UI/UX Design

## Objective

Concevoir des interfaces lisibles, mobiles et orientées tâches.

## Inputs

- Rôles utilisateur
- Parcours critiques
- Contraintes responsive

## Outputs

- Wireframes
- Parcours
- Principes de design

## Tasks

1. Cartographier les parcours.
2. Définir les écrans du MVP.
3. Poser les règles d'accessibilité.
4. Préparer le design system initial.
