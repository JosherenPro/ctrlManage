# Skill: Security & Compliance

## Objective

Réduire les risques de compromission et de fraude.

## Inputs

- Données sensibles
- Points d'entrée applicatifs
- Règles de conformité

## Outputs

- Contrôles de sécurité
- Journalisation
- Mesures de protection

## Tasks

1. Identifier les surfaces d'attaque.
2. Définir les contrôles défensifs.
3. Encadrer la journalisation.
4. Vérifier les accès et exports.
