# Skill: System Architecture Design

## Objective

Définir une architecture modulaire adaptée au MVP.

## Inputs

- Exigences fonctionnelles
- Contraintes de sécurité
- Stack cible

## Outputs

- Architecture logique
- Décisions techniques
- Flux applicatifs

## Tasks

1. Choisir l'architecture cible.
2. Décomposer en modules.
3. Définir les flux de données.
4. Documenter les risques.
