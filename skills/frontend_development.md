# Skill: Frontend Development

## Objective

Construire l'interface de gestion académique par rôle.

## Inputs

- Parcours UX
- Contrats API
- Rôles utilisateur

## Outputs

- Pages
- Composants
- États de chargement et d'erreur

## Tasks

1. Définir la structure applicative.
2. Développer les dashboards.
3. Ajouter les formulaires et vues de données.
4. Connecter les appels API.

- `frontend/`
