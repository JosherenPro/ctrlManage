# Skill: DevOps & CI/CD

## Objective

Automatiser la qualité et préparer le déploiement continu.

## Inputs

- Structure du dépôt
- Pipeline cible
- Environnements de déploiement

## Outputs

- Workflow CI
- Conteneurisation
- Standards d'environnement

## Tasks

1. Définir les jobs de validation.
2. Préparer les artefacts Docker.
3. Formaliser les variables d'environnement.
4. Rendre le pipeline extensible.

## Deliverables

- `.github/workflows/ci.yml`
- `Dockerfile`
