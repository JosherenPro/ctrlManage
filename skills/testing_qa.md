# Skill: Testing & Quality Assurance

## Objective

Sécuriser les flux critiques par tests automatisés.

## Inputs

- User stories
- Critères d'acceptation
- Risques fonctionnels

## Outputs

- Stratégie de tests
- Jeux de test
- Couverture des parcours critiques

## Tasks

1. Définir la pyramide de tests.
2. Cibler les flux à haut risque.
3. Automatiser les tests unitaires, intégration et E2E.
4. Intégrer l'exécution en CI.

## Deliverables

- `tests/`
